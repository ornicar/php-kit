\PrismicApi

This class embodies a connection to your prismic.io repository's API.

Initialize it with Prismic\Api::get(), and use your Prismic\Api::forms() to make API calls (read more in the kit's README file)

bookmark()

bookmark(string  $name) : string

From a bookmark name, returns the ID of the attached document.

You can then use this ID for anything, for instance to query with a predicate that looks like this [:d = at(document.id, "abcdefghijkl")]. Most starter projects embed a helper to query a document from their ID string, which makes this even easier.

Parameters

string $name

the bookmark name to use

Returns

string —

the ID string for a given bookmark name

forms()

forms() : \stdClass

Returns all forms of type Prismic\SearchForm that are available for this repository's API.

The intended syntax of a call is: api->forms()->everything->query(query)->ref(ref)->submit(). Learn more about those keywords in prismic.io's documentation on our developers' portal.

\stdClass —

get()

get(string  $action, string  $accessToken = null, \Guzzle\Http\ClientInterface  $client = null, \Prismic\Cache\CacheInterface  $cache = null) : \Prismic\Api

This is the endpoint to build your API, and is a static method.

If your API is set to "public" or "open", you can instantiate your Api object just like this: Api::get('http://idofyourrepository.prismic.io/api')

Parameters

string $action

the URL of your repository API's endpoint
string $accessToken

a permanent access token to use to access your content, for instance if your repository API is set to private
\Guzzle\Http\ClientInterface $client

by default, the client is a Guzzle with a certain configuration, but you can override it here
\Prismic\Cache\CacheInterface $cache

Cache implementation

\RuntimeException

\Prismic\Api —

the Api object, useable to perform queries
