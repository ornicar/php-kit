\Prismic

\PrismicResponse

This class embodies a search request response from the prismic.io API, containing a list of documents, as well as meta-information about this list (mostly about pagination).

This is what is returned when you call submit() on a search request, which is the only kind of request except for the API endpoint. So, let's say that when you call prismic.io's API, you get that object a lot.

Do remember that requests in prismic.io are paginated by 20 by default.
