\Prismic

\PrismicLinkedDocument

This class embodies the linked documents that are attached to a given documents.

Each time prismic.io returns a document to a query, it also gathers in an array the characteristics of all the documents it links to. These links can either be link fragments, or hyperlinks in StructuredText fragments, ... anything within the current document.

The purpose of this feature is to make it trivial to develop a content discovery feature that has a "Linked in this article" box.
